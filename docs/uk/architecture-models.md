# Архітектура Model-as-Schema: UI Компоненти

Цей документ класифікує усі доменні UI-моделі доступні в екосистемі `@nan0web/ui`, вказуючи версію їх появи та призначення. Цей звід допомагає мовним моделям та агентам швидко розуміти наявний інвентар без читання вихідного коду.

## Реєстр Компонентів

| Назва Компонента | Версія | Короткий Опис |
|---|---|---|
| **Базова Взаємодія та Форми** |
| `InputModel` | v1.0.0 | Стандартне введення тексту та скалярних значень. |
| `ButtonModel` | v1.0.0 | Кнопки-тригери для ініціалізації намірів корситувача. |
| `SelectModel` | v1.2.0 | Інтерфейси випадаючих списків. |
| `AutocompleteModel` | v1.2.0 | Поле вводу з живим пошуком та автодоповненням варіантів. |
| **Візуалізація Даних** |
| `TableModel` | v1.0.0 | Таблиці баз даних, які монтують ряди на інші моделі. |
| `TreeModel` | v1.3.0 | Ієрархічне відображення вкладених структур та директорій. |
| `BreadcrumbModel` | v1.7.0 | Навігаційні шляхи для відображення глибини контексту. |
| **Зворотній Зв'язок (Feedback)** |
| `ToastModel` | v1.0.0 | Тимчасові сповіщення, що зникають самі. |
| `ConfirmModel` | v1.0.0 | Запити підтвердження деструктивних дій. |
| `SpinnerModel` | v1.2.0 | Індикатор завантаження багатоетапних дій. |
| **Верстка Контенту** |
| `FAQModel` | v1.6.0 | Відображення секцій з частими запитаннями та відповідями. |
| `FeatureGridModel`| v1.6.0 | Сітка переваг або фіч для промо-блоків. |
| `CommentModel` | v1.6.0 | Базова логіка коментарів або обговорень. |
| **The Domain Bloom (Додано у v1.10.0)** |
| `HeaderModel` | v1.10.0 | Головний верхній навігаційний блок (шапка). |
| `FooterModel` | v1.10.0 | Нижній блок сторінки (юридичні лінки, мапа сайту). |
| `HeroModel` | v1.10.0 | Головна секція привертання уваги з Call-to-action. |
| `AccordionModel` | v1.10.0 | Групи даних, які згортаються (розширює логіку FAQ). |
| `TabsModel` | v1.10.0 | Сегментація інтерфейсу через вкладки (таби). |
| `GalleryModel` | v1.10.0 | Сітки для відображення зображень чи візуальних активів. |
| `PricingModel` | v1.10.0 | Відображення тарифів та матриць підписок. |
| `StatsModel` | v1.10.0 | Блоки для цифрової статистики чи метрик проекту. |
| `TimelineModel` | v1.10.0 | Хронологічна лінія часу для відображення історії дій. |
| `TestimonialModel` | v1.10.0 | Використовує CommentModel для відгуків клієнтів. |
| `EmptyStateModel` | v1.10.0 | Інтерфейс-заглушка при відсутності даних для відображення. |
| `BannerModel` | v1.10.0 | Глобальні критичні сповіщення поверх всього інтерфейсу. |
| `ProfileDropdownModel`| v1.10.0 | Стандартне меню опцій користувача для авторизованої зони. |
| **Системна Оркестрація** |
| `LayoutModel` | v1.0.0 | Кореневий інтерфейс додатка (Head, Body, Sidebar). |
| `ShellModel` | v1.0.0 | Оркестратор OLMUI-маршрутизації та логіки вкладень. |
| `SandboxModel` | v1.7.0 | Ізольоване середовище для візуального тестування (Playwright). |
